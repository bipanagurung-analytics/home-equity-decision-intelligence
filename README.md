> **Project Goal:** Build a decision-support solution that helps lenders identify risk earlier, understand borrower behavior, and make more informed credit decisions.

---

# Why This Project?

Financial institutions process thousands of loan applications every year, yet incomplete borrower information and inconsistent underwriting continue to increase lending risk. This project demonstrates how machine learning can transform raw borrower data into explainable insights that support faster, more consistent, and data-driven lending decisions.

# What the Data Revealed

## Hidden Risk

Borrowers who did not disclose their **Debt-to-Income ratio** were substantially more likely to default, revealing that missing financial information itself carried predictive value.

## Past Behavior Predicts Future Risk

<img width="1086" height="568" alt="image" src="https://github.com/user-attachments/assets/5f4c17fd-bee0-4adb-8c7d-d0cfd1ccfac2" />


Borrowers with more delinquent credit accounts consistently exhibited higher default rates, making repayment history one of the strongest indicators of future lending risk.

## Different Borrowers Require Different Lending Strategies

<img width="1089" height="620" alt="image" src="https://github.com/user-attachments/assets/ae1f1582-2117-402e-9bd9-7c44f4a09b68" />


Clustering revealed distinct borrower groups with different credit histories, debt burdens, and delinquency patterns, suggesting that a one-size-fits-all lending strategy may overlook meaningful differences in borrower risk.

# Did It Work?

<!-- INSERT IMAGE: ROC Curve -->

Seven machine learning models were evaluated using cross-validation and independent testing. **Random Forest** delivered the strongest overall predictive performance while maintaining excellent predictive capability and model interpretability.

---

# Why the Model Made Those Decisions

<!-- INSERT IMAGE: Random Forest Variable Importance -->

Rather than functioning as a black box, the model highlighted the key factors influencing loan default, including missing Debt-to-Income information, delinquency history, derogatory reports, and credit history.

# Key Takeaways

- Identified high-risk borrowers before loan approval.
- Estimated missing borrower financial information.
- Explained the primary drivers behind loan default.
- Segmented borrowers into meaningful financial profiles.
- Demonstrated how machine learning can support more consistent lending decisions.
